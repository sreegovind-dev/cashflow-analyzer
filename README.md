# Cashflow Analyzer 💰

A web-based cashflow analysis application built using Flask, designed to analyze financial data and predict cashflow risks using Machine Learning.

---

## 🚀 Features

* 📊 Revenue comparison visualization
* 📉 Cashflow tracking and analysis
* 🤖 Machine Learning model for predicting cashflow issues
* 🔐 Simple login system
* 📈 ROC Curve & Confusion Matrix visualization

---

## 🛠️ Tech Stack

* Python
* Flask
* Pandas, NumPy
* Matplotlib, Seaborn
* Scikit-learn

---

## ▶️ How to Run

1. Clone the repository
2. Install dependencies:

```
pip install flask pandas numpy matplotlib seaborn scikit-learn
```

3. Run the app:

```
python Appy2.0.py
```

4. Open browser:

```
http://127.0.0.1:5000
```
